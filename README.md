# Skatakia Project

Skatakia is a nonsense project for experimenting with [Github](https://github.com/) capabilities.

## Compile

Use the gcc compiler to compile the project. Type the following to compile:

```bash
make
```

## Usage

```bash
./bin/skata
```
or
```bash
cd bin
./skata
```

to execute.
